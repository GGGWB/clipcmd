# 贡献者公约(简版)

本项目遵循简化的 Contributor Covenant。参与本项目(包括 Issue、PR、讨论)即代表你同意以下准则。

## 我们的承诺

为了营造一个开放友好的环境,作为贡献者和维护者,我们承诺:无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份,参与本项目的人都不会遭受骚扰。

## 我们的准则

**鼓励的行为:**

- 展现同理心和善意
- 尊重不同观点和经验
- 给予建设性反馈,也优雅地接受反馈
- 承担责任,向受我们错误影响的人道歉,并从中学习

**不可接受的行为:**

- 使用性化语言或 imagery、性关注或挑逗
- 钓鱼、辱骂或贬损的评论;人身或政治攻击
- 公开或私下骚扰他人
- 未经明确许可泄露他人私人信息
- 其他可被合理认定为不适合专业环境的行为

## 执行

违反准则的行为可通过 Issue 或邮件报告给维护者。投诉会保密处理,违规者可能被警告、临时或永久封禁。

## 出处

本公约改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版,简化以适应小项目。
